1. 环境配置

工具链安装

# 指定版本工具链
rustup toolchain install nightly-2024-03-05
rustup default nightly-2024-03-05
rustup target add riscv64gc-unknown-none-elf

# 验证安装
cargo --version
rustc --version

qemu模拟器安装

Arch上安装qemu模拟器比较简单，运行下述命令即可：

sudo pacman -S qemu-system-riscv

sequenceDiagram
    participant BIOS
    participant Bootloader
    participant Kernel
    participant SBI
    
    BIOS->>Bootloader: 加载并执行
    Bootloader->>SBI: 初始化硬件

    Kernel->>Kernel: 初始化基本环境
    Kernel->>SBI: 关机请求
    SBI->>Hardware: 系统关机
